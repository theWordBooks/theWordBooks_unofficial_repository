Font Recommendation:

Install Cardo version 104 which is embedded in the module. If you don't already have it installed it should download into theWord > Fonts from their please install it into your system. This will help the Hebrew to display correctly.
If for some reason you can't find it on your system it is available here: https://scholarsfonts.net/cardofnt.html#DownloadLink

theWord Features:

This module contains the BDB entries listed both by their Strong’s numbers and by the order of their headwords as listed within the BDB lexicon.

  • Dictionary topics
    • Strong's Number Index: H9
    • BDB Hebrew headword Index: אֲבֵדָה
  • Verse popups
  • Fully searchable text
  • Easy navigation of topics via topics tree display.
  • Linked to Bible view via Lemma (BDB headwords), and Strong's Numbers
  • Special Text Colors
    • Normal: Text
    • Hyperlink: Gen. 9:8
    • Strong's Numbers: H9
    • Hebrew:  אֲבֵדָה


Please note Work in Progress on this module

There may be a difference between the versification of the uBDB and theWORD Bible text. This may be due to LXX or BHS having different versification at times.

Arabic words are not yet added to the source.

Some incorrect references have been removed. 

There may be some Hebrew sentences that are scrambled. 

The tables for verbs may at times be on the wrong spot. Please note this is a work in progress.
