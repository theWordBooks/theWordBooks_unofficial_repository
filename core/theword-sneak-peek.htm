theWord Sneak-peek version

Important: Always keep backups of theWord before using a Beta version. Note the beta has gone through a lot of testing but each user is responsible for their own theWord backups.

Step 1: theWord 3-6 Upgrade

theWord 6 or previous version use: theWord Installer (else go to Step 2)

Step 2: Download ''theWord Changelog from Official Repository''

Important: Read the changelog in book form from Official Repository or changelog.txt which will be installed theWord main directory.

Step 3. Now you can download Sneak-peek version.

If you did not do this in this order of 1, 2, 3 the Green check in Add Titles will show up but it will not work because the file mentioned above as "theWord Installer " must be installed first or everything else will not work!


twsneak-peek is the sneak-peek version of theWord where the new features are first added and become available for early testing. Please only discuss twsneak-peek in the github and not in theWord forum. The forum is for theWord beta and beyond. If you are unable to access the github please request permission at twalpha@theword.net but first make sure to read the rest of this book and setup a github account. Then when you request permission include the email used for your github account. 

This book contains information on how to use theWord Alpha version, report bugs, request features, and the feature list.
