twsneak-peek is the sneak-peek version of theWord where the new features are first added and become available for early testing. Please only discuss twsneak-peek in the github and not in theWord forum. The forum is for theWord beta and beyond. If you are unable to access the github please request permission at twalpha@theword.net but first make sure to read the rest of this book and setup a github account. Then when you request permission include the email used for your github account. 

This book contains information on how to use theWord Alpha version, report bugs, request features, and the feature list.
